# Alarm Clock

A simple alarm clock application built with Python and Tkinter.

## Features

* Set an alarm using the HH:MM format
* Time format validation
* Start and Stop alarm buttons
* Choose your own MP3 or WAV alarm sound
* Automatic alarm trigger at the selected time

## Technologies

* Python
* Tkinter
* Datetime
* File Dialog

## How to Run

1. Install Python.
2. Run the file:

python main.py
